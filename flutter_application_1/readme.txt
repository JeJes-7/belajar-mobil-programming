Ini adalah source code dasar untuk belajar mobile programming menggunakan Flutter.

Di dalam repositori ini, saya hanya mengunggah bagian yang mengalami perubahan dari template Flutter standar, yaitu folder lib/ dan file pubspec.yaml saja. Sementara bagian lainnya tetap menggunakan struktur bawaan Flutter.

Untuk menjalankan aplikasinya, buka terminal dan jalankan perintah "flutter run"
